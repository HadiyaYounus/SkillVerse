# SkillVerse 

SkillVerse is a modern learning platform built using React and Express.js. It allows users to explore programming skills, search courses, filter by level, and view detailed information about each skill.

## Features

- Modern dark UI
- Responsive design
- React Router navigation
- Search skills
- Filter by level
- Dynamic skill details page
- About page
- Contact page
- Custom 404 page
- Express.js backend
- JSON-based data storage
- Axios API integration

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js

### Data
- JSON

## Installation

### Clone repository

```bash
git clone <repository-url>
```

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm start
```
